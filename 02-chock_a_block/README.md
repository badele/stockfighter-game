# Level 1

## Usage

Usage:
  level1 [-q=<qorder> | --qty=<qorder>] [-l=<nb> | --loop-size=<nb>]
  level1 -h | --help

Arguments:

Options:
  -l=<nb> --loop-size=<nb>          Loop number [default: 10000]
  -q=<qorder> --qty=<qorder>        Stock quantity must be order [default: 100000]
  -h --help                         Help usage
